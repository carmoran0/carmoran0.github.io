Ventana Win9x con barra de título degradada — el contenedor base de toda superficie de CarlOS.

```jsx
<Window title="🎵 música — reproductor.exe" width={480} focused>
  <WindowInset mono>84.201 scrobbles</WindowInset>
</Window>
```

- `variant`: `active` (azul, defecto) · `inactive` (gris) · `purple` (mastodon/social).
- Título siempre con emoji inicial y nombre `.exe` o ruta `carpeta\`.
- `WindowInset` para el área blanca hundida (listas, logs, texto).
